【카테고리】 웹개발 기초지식
【제목】 랜딩페이지 vs 홈페이지, 뭐가 다를까? 쉽게 정리

"랜딩페이지 만들어주세요" "홈페이지 만들어주세요"
이 두 가지, 뭐가 다른 걸까요?

결론부터 말하면, 목적이 다릅니다.


■ 랜딩페이지란?

하나의 목적에 집중하는 1페이지짜리 사이트입니다.

예시:
- 신제품 출시 → "지금 사전예약하세요"
- 이벤트 홍보 → "참여하고 경품 받기"
- 앱 다운로드 → "지금 다운로드"

특징:
- 보통 1~3페이지
- 하나의 CTA(행동 유도) 버튼에 집중
- 스크롤하면서 정보를 순서대로 보여줌
- 제작 비용: 15만원~50만원


■ 홈페이지란?

회사/브랜드의 전체 정보를 담는 사이트입니다.

예시:
- 회사 소개, 서비스 안내, 팀 소개
- 포트폴리오, 고객 후기
- 문의 폼, 오시는 길

특징:
- 보통 5~10페이지 이상
- 여러 메뉴와 하위 페이지
- 회원가입, 게시판 등 기능 포함 가능
- 제작 비용: 40만원~150만원


■ 어떤 걸 만들어야 할까?

| 상황 | 추천 |
| 특정 제품/서비스 홍보 | 랜딩페이지 |
| 사업 시작, 회사 소개 필요 | 홈페이지 |
| 광고 연결용 페이지 | 랜딩페이지 |
| 고객이 검색해서 찾아오게 | 홈페이지 |
| 빠르게, 저렴하게 | 랜딩페이지 |

사실 요즘은 랜딩페이지 스타일로 만드는 홈페이지도 많습니다.
깔끔한 원페이지에 필요한 정보를 다 담는 방식이죠.
